# cube-ui-eleme-project

Vue2.5 + cube-ui 实现饿了么 WebApp:
vue-cli 初始化项目、mock 数据；使用了 cube-ui 开源的七个组件和一个模块。
以外卖商家页面为主线， 开发商品详情部分（商家详情弹层、滚动列表浏览商品、添加购物车 及小球下落动画、查看购物车详情、结算）、 商家介绍部分（商家详情、查看评分、收藏商家）、评论 （过滤显示评论）等模块。
